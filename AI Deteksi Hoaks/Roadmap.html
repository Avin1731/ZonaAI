Roadmap Proyek VeritasAI

Fase Aktivitas Utama Output Teknis
1. Inisiasi & Desain Prompt Definisi branding, riset gaya bahasa hoaks, dan perancangan "System Prompt" untuk Gemini API. Dokumen Spesifikasi & Prompt Template.
2. Core Engine (Backend & RAG) Membangun API dengan FastAPI, integrasi Search API (Serper/Google) untuk mencari fakta real-time. Backend Service (FastAPI) & RAG Module.
3. Pengembangan Frontend Pembuatan Landing Page, form input berita, dan dashboard visualisasi skor (Gauge Chart). Frontend Next.js (Dashboard & Landing Page).
4. Integrasi Memori & Feedback Koneksi ke Supabase untuk menyimpan riwayat scan dan fitur koreksi user (Feedback Loop). Database Schema & User History Log.
5. Finalisasi & Pitching Uji akurasi deteksi, optimasi kecepatan respons, dan penyusunan paper/materi presentasi. Final Product & Demo Video.

Flowchart Arsitektur VeritasAI

Start: User Input (Link/Teks/Snippet)
Module 1: Web Scraper & News Search API (Cari Fakta)
Module 2: RAG Prompting (Kirim Teks + Fakta ke Gemini)
Module 3: Probabilistic Scoring (Hitung Skor 1-100)
Output: Tampilkan Skor, Rujukan & Edukasi
Storage: Simpan ke Supabase & Terima Feedback User
