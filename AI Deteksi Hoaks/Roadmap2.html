Timeline Pengerjaan

  • Fase 1: Research & Prompt Engineering (Minggu 1)
    Riset pola hoaks terbaru dan penyusunan instruksi (System Prompt) untuk AI agar output konsisten dalam format JSON.
  • Fase 2: Backend & Search Integration (Minggu 2)
    Membangun server FastAPI dan mengintegrasikan Serper API untuk mengambil data berita real-time sebagai referensi AI.
  • Fase 3: Frontend & Visualization (Minggu 3)
    Develop UI menggunakan Next.js, pembuatan gauge chart untuk visualisasi skor, dan halaman landing page produk.
  • Fase 4: Database & Feedback System (Minggu 4)
    Implementasi Supabase untuk menyimpan riwayat pencarian pengguna dan fitur feedback "Koreksi Fakta".
  • Fase 5: Testing & Paper Production (Minggu 5)
    Uji coba sistem dengan berbagai jenis berita, dokumentasi teknis, dan finalisasi materi presentasi lomba.
